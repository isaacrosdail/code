# css-practice
A collection of HTML templates and structured practice exercises for mastering CSS. Includes card layouts, navigation bars, hero sections, grids, forms, and page layouts with a clean git workflow for iterative learning. Perfect for building muscle memory with modern CSS patterns through hands-on practice sessions.

Note to self:
- Since we're using basic HTML/CSS only, just use Live Server extension while editing. Right-click HTML file and open with live server.

**Practice Session Workflow**
1. Start a session: `git switch -c session-DD.MM.YY`
2. Choose a template & copy it to practice sessions: `cp templates.card.html practice-sessions/session1.html`
3. Practice for 30 mins - Style the template
4. Save a screenshot of result
5. Reset for next session: `git switch main` then `git branch -D session-MM.DD.YY`

**Practice Ideas**
- Cards: GitHub profile cards, product cards, pricing cards
- Navbars: GitHub navbar, portfolio site navigation, mobile-first nav
- Heroes: Landing page heroes, portfolio headers, app splash screens
- Grids: Photo galleries, dashboard layouts, blog post grids
- Forms: Login forms, contact forms, multi-step forms
- Layouts: Blog layouts, portfolio layouts, admin dashboards
