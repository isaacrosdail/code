With :focus-visible() on the parent:

Here, focusing an input does not change the outline color. Clicking the input gives focus to the input element, not the parent .field div. Since :focus-visible only applies when the element itself is focused, the selector does not match and no styles are applied.

With :focus-within() on the parent:

Here, the outline does change using :focus-within. When the input is focused, the parent .field-two div matches :focus-within because one of its descendants has focus, so the styles apply.
